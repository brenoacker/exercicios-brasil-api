1) Baixar o projeto
2) Excluir a pasta "venv"
3) Criar um ambiente virtual -> python -m venv ./venv
4) Ativar o ambiente virtual executando o arquivo Active.ps1 em -> venv/Scripts/Active.ps1
5) Instalar os módulos do arquito requirements.txt -> pip install -r requirements.txt
6) Subir o servidor -> python manage.py runserver
7) Para fechar o servidor -> crtl c
